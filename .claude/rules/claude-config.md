# Claude Configuration Conventions

## Language Policy

- UI messages and console output, code comments, test names, and commit messages must be written in Japanese
- Rule files (`.claude/rules/`, `docs/rules/`) and skill files (`.claude/skills/`) must be written in English
- This includes frontmatter (`name`, `description`), headings, and body content
